# Frontend

React + Vite + TypeScript dashboard for Quicklook.

## Setup

```bash
npm install
```

## Run (dev)

```bash
npm run dev -- --host 0.0.0.0 --port 5173
```

## Build

```bash
npm run build
```

## Configuration

Set the backend URL via `VITE_BACKEND_URL` (default `http://localhost:8000`).
